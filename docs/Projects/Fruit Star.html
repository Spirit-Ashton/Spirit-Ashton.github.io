Fruit Star!

• Lua Based Platformer
Fruit Star is a project focused on creating a more stable platformer with solid foundations.
Apples, Apples, Apples!
This Project features collectables which are generated using map tiles, each individually collectable and adding to a tally!
What doesn't kill you...
The Health System Revolves around these hearts, which are procedurally generated by a queue system and can be any amount!
With multiple Levels, Collectables, Lives and Enemy CPU, I've set a strong platform to build off of.
Angry Enemies!
The Enemies were built to have 3 stages, activated when jumped on. Each stage is harder to dodge than the last!
Smart Pathing
Enemies have collision detectors so that they can avoid dropping into bottomless pits, aslong as they arent blistering with rage!
Bouncy Heads!
Bouncing on Enemies give the player an extra boost in height! The Enemies also have smart deactivation protocols for efficiency aswell as reload distances!
Fruit Star enabled me to solidify my skills and employ more techniques of game design into my code.
